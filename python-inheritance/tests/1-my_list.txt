The ``MyList`` class inherits from ``list``
==========================================

Check if MyList inherits from list:
    >>> MyList = __import__('1-my_list').MyList
    >>> issubclass(MyList, list)
    True

Check sorted print:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Check original list is still the same:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Check with negative numbers:
    >>> my_list = MyList([-1, -5, 0, 10])
    >>> my_list.print_sorted()
    [-5, -1, 0, 10]
Check with empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []
